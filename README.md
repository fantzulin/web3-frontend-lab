# Web3 Frontend Lab - Sa (卅)

## Built incrementally, this 30-day challenge showcases progressive Web3 frontend features, forming a complete project by the end.

A 30-day Web3 frontend challenge.  
Each day is represented using **ancient Chinese numerals**: 壹, 貳, 參, …, 卅.  
This project is designed to progressively build a complete Web3 frontend system.

---

## Usage

- Each day can be a separate **page** (`pages/sa1.tsx`, `pages/sa2.tsx`, …) or **component**.  
- The sidebar or navigation can list the days using **ancient Chinese numerals** (壹, 貳, 參, …, 卅).  
- By the end of the 30 days, all features are integrated into a **complete Web3 frontend lab**.
